## Local Development

### Prerequisites
- Node.js 16+
- Python 3.8+
- MongoDB

### Frontend Setup
```bash
cd frontend
yarn install
yarn start
```

### Backend Setup
```bash
cd backend
pip install -r requirements.txt
uvicorn server:app --reload --port 8001
```
